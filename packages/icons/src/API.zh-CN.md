### DynamicIcon Props

> 继承自 `HTMLAttributes<HTMLSpanElement>`

| 属性名 | 描述 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| name | - | `string` | - |
| variant | - | `'outlined' \| 'filled' \| 'two-tone' \| 'multi-color'` | - |
| strokeWidth | - | `number` | - |
| strokeLinecap | - | `'butt' \| 'round' \| 'square'` | - |
| strokeLinejoin | - | `'miter' \| 'round' \| 'bevel'` | - |
| fill | - | `string \| string[]` | - |
| spin | - | `boolean` | - |

### Checkbox Props

> 继承自 `Omit<HTMLAttributes<HTMLLabelElement>, 'onChange' | 'defaultValue'>`

| 属性名 | 描述 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| size | - | `CheckboxSize` | - |
| checked | - | `boolean` | - |
| defaultChecked | - | `boolean` | - |
| value | - | `boolean` | - |
| defaultValue | - | `boolean` | - |
| indeterminate | - | `boolean` | - |
| disabled | - | `boolean` | - |
| onChange | - | `(checked: boolean) => void` | - |
